This code computes the knight tour for chessboard of sizes 5 * 5 onwards. The input is stdin and the output is written to a file called output.txt. The output.txt gets created in the pwd everytime the code is run. 

Instructions for running the code : python 671_hw_2a.py and then enter the input size . 

eg input 8 for a 8 * 8 code. 
Input 20 for a 20 * 20 code.
